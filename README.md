# Hutch-curse
Code for "Assisting gene flow  in a complex and changing world despite Hutchinson’s curse"

Assisted Migration Model Parallel.Rmd – As written, this conducts one simulation of each possible combination of parameters (landscape type, migration, selection) and saves it. We reran this 30 times.
DataCleaning_NoRescue.Rmd – combines the outputs and shapes data for the no rescue plot ready (Figure 1)
DataCleaning_Rescue.Rmd – shapes the data for the rescue plots
Plots_NoRescue.Rmd – Creates Figure 2
Plots_Rescue.Rmd – Creates the remaining Figures 3-4
Regressions.Rmd – Runs the regressions
